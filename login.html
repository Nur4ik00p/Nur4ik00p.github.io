Главная / Примеры

Примеры использования

Ниже приведены практические примеры использования нашего API в различных сценариях.

Пример 1: Базовое использование

Простой пример инициализации и использования основных функций:

javascript 
import { init } from '@project/core';

// Инициализация с базовыми настройками
const app = init({
  apiKey: 'your-api-key',
  debug: true
});

// Использование API
app.getData()
  .then(result => console.log(result))
  .catch(error => console.error(error));

Пример 2: Работа с асинхронными функциями

Пример использования асинхронных функций для обработки данных:

javascript 
import { init, processData } from '@project/core';

async function main() {
  // Инициализация приложения
  const app = init({
    apiKey: 'your-api-key',
    debug: false
  });
  
  try {
    // Получение данных
    const rawData = await app.getData();
    
    // Обработка данных
    const processedData = await processData(rawData, {
      filter: true,
      normalize: true
    });
    
    // Отправка результатов
    await app.sendResults(processedData);
    console.log('Обработка завершена успешно');
  } catch (error) {
    console.error('Произошла ошибка:', error);
  }
}

main();
